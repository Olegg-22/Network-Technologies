# Змейка по сети (Multiplayer Snake Game)

Многопользовательская игра "Змейка" с поддержкой сетевого взаимодействия по протоколу UDP.

## Описание

Классическая игра "Змейка" с возможностью играть по сети с другими игроками. Управляя своей змейкой, нужно поедать еду и расти, избегая столкновений со своим телом и змейками других игроков.

### Особенности
- **Peer-to-peer архитектура** с топологией "звезда"
- **Автоматическое обнаружение игр** через multicast (239.192.0.4:9192)
- **Отказоустойчивость**: при отключении мастера его роль переходит к заместителю (Deputy)
- **Совместимость** с другими реализациями протокола snakes.proto

## Сборка и запуск

### С помощью Makefile

```bash
# Сгенерировать proto файлы и собрать
make build

# Запустить игру
make run

# Очистить
make clean
```

### Вручную

```bash
# Собрать
go build -o bin/snake ./cmd/snake

# Запустить
./bin/snake
```
