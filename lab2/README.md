# Передача файла по TCP с подсчётом скорости передачи данных

Приложение предназначено для передачи произвольного файла от клиента к серверу по TCP.  
Сервер принимает файл, сохраняет его в каталог `uploads/` и каждые 3 секунды выводит в консоль **мгновенную** и **среднюю** скорость приёма данных** для каждого активного клиента.

---

## Состав проекта

- **Server/** — серверная часть, принимает файл и считает скорость передачи.  
- **Client/** — клиентская часть, отправляет файл на сервер.

---

## Запуск

### Сервер
```bash
go run Server/main.go <порт_сервера>
```

### Клиент
```bash
go run Client/main.go <путь_к_файлу> <ip_сервера> <порт_сервера>
```

**Пример:**
```bash
go run Client.go ./data.txt 127.0.0.1 9000
```
