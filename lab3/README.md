# Асинхронное приложение для поиска локаций, погоды и достопримечательностей

Приложение позволяет асинхронно взаимодействовать с несколькими внешними API для поиска информации о локациях, погоде и интересных местах.

## Рабочий процесс

1. **Поиск локаций** - пользователь вводит название места
2. **Выбор локации** - пользователь выбирает из списка найденных вариантов
3. **Параллельные запросы**:
   - **Погода** для выбранной локации
   - **Интересные места** для каждого места с их описанием
4. Все данные собираются в **единый ответ**

## Запуск

```bash
go run main.go
```
